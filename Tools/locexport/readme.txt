+-----------------------------------------------+
| Author: LondonCity                            |

+-----------------------------------------------+

Run this simple PowerShell skript within a NFSCO
localization directory (e.g. C:\git\nfscoloc\DE\)
for autoamtically exporting your translation to
UTF-8 formatted Labrune dump files. Afterwards,
you can import the translation to the language
binary files using Labrune.v2.0.0.100.
