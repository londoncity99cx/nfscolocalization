+-----------------------------------------------+

This easy-to-use PowerShell tool helps you with
comparing two Labrune Dump files with each other.
This is useful for identifying changes made in
newer NFSCO language files. Please note, that you
have to export the data from the langauge file
you want to compare first with Labrune to a Labr-
une Dump file.

Features:
- tells you if the files are identical (by hash).
- shows comparison results in a RichTextBox.
- highlights changed strings in red and green.
- option to save comparison results to .txt-file.
- SplitView for looking at both language files at
  the same time.
