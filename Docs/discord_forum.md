This post is all about the localization of NFSCO. *Please note, that as of right now this is no official localization & that the available translations are fan made.*

Since NFSCO describes itself as "The definitive Need for Speed: Carbon experience", the available translations reflect that concept of authenticity of the game & will blend in seamlessly. All reward cards, Challenge Series events & every other NFSCO-related string has been translated carefully. Some strings from the base game language files that contained spelling errors in the UI were corrected too.

## **HOW TO INSTALL CONVERSION PACK (RECOMMENDED)**
1. Create a backup of your installation.
2. Find the download link of the desired localization pack.
3. Download the *full conversion pack* (e.g. nfsco_de_conversion.zip).
4. Extract the .zip-file to your NFSCO install. If asked to replace files, select "replace all".
5. Launch the game & have fun!

To revert to English, download the conversion pack from :flag_um: & extract it over your install. It contains the unmodified localization files from NFSCO.

## **HOW TO INSTALL LANGUAGE FILES SEPERATELY**
1. Create a backup of your installation.
2. Find the download link of the desired localization pack.
3. Download the .zip-files you want (e.g. nfsco_de_movies.zip).
4. Extract the .zip-file to your NFSCO install. If asked to replace files, select "replace all".
5. Change the language by editing the games registry file (c:\yourgame\NFSCO\ registry.json).
