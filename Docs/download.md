# Download Instructions

### HOW TO INSTALL CONVERSION PACK (RECOMMENDED)
1. Create a backup of your installation.
2. Find the download link of the desired localization pack.
3. Download the *full conversion pack* (e.g. nfsco_de_conversion.zip).
4. Extract the .zip-file to your NFSCO install. If asked to replace files, select "replace all".
5. Launch the game & have fun!

To revert to English, download [this](https://1drv.ms/f/c/e8ab278086f1150c/IgAPwog7qlR7Trt50RvcPW06AXSdCjHDvpLWMN0KnwSk6Qc) conversion pack containing and extract it over your install. It contains the unmodified localization files from NFSCO.

### HOW TO INSTALL LANGUAGE FILES SEPERATELY
1. Create a backup of your installation.
2. Find the download link of the desired localization pack.
3. Download the .zip-files you want (e.g. nfsco_de_movies.zip).
4. Extract the .zip-file to your NFSCO install. If asked to replace files, select "replace all".
5. Change the language by editing the games registry file (c:\yourgame\NFSCO\ registry.json).
